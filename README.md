Our mission is to transform self-improvement from an isolated effort into a structured, rewarding, and sustainable experience accessible to everyone.

To become a globally recognized AI-native ecosystem company that powers the future of personal growth, learning, productivity, creator development, and digital self-improvement.

### The Story Behind Provolution Technologies

Provolution Technologies was founded on a simple observation: **Modern technology is exceptionally good at capturing attention but often fails at helping people grow.**

People use separate applications for studying, fitness, productivity, habit tracking, content creation, and community engagement. This fragmentation creates friction, inconsistency, and abandoned goals. Many individuals start ambitious journeys but struggle to maintain momentum because they lack systems that connect progress across different areas of life.

The rise of artificial intelligence presented an opportunity to rethink this approach. Instead of creating another standalone application, the vision was to build an ecosystem where learning, fitness, productivity, creativity, and accountability work together through intelligent systems and AI assistance. This vision became the foundation of Provolution Technologies Pvt Ltd.

The company's philosophy is built around one belief: **Growth is not about perfection. Growth is about consistency.**

Every product developed under Provolution Technologies is designed to help users build that consistency through intelligent guidance, visible progress tracking, gamification, and supportive communities.

## About SRE

**SRE — Start · Restart · Explore** is the flagship platform developed by Provolution Technologies Pvt Ltd.

It was created to solve a problem faced by millions of people: **Starting is difficult. Restarting is even harder.**

Many people begin learning new skills, fitness routines, creative projects, or productivity habits with enthusiasm. However, life interruptions, setbacks, and inconsistency often cause progress to stop. Traditional platforms rarely support users when they fall behind, making them feel like they have failed.

SRE was built on a different philosophy. The platform recognizes that setbacks are a natural part of growth and that restarting should be encouraged rather than punished.

The name SRE represents three core principles:

### Start

Every achievement begins with a first step. Whether someone wants to learn a new skill, improve their fitness, create content, develop a business, or build better habits, SRE provides the tools and structure needed to begin.

### Restart

Progress is rarely linear. SRE embraces the idea that restarting is not failure. Life happens, goals get interrupted, and routines break. The platform is designed to help users return to their journey without judgment while preserving their progress and momentum.

### Explore

Growth thrives on curiosity. SRE encourages users to discover new interests, learn new skills, connect with communities, and expand beyond their comfort zones. Exploration is treated as an essential part of personal development.

### Mission of SRE

To create a unified AI-powered self-growth platform that helps individuals build consistency across learning, fitness, productivity, content creation, and personal development through gamification, accountability, and intelligent guidance.

### Vision of SRE

To become the world's leading AI-native self-growth ecosystem where people can track progress, build discipline, discover opportunities, and continuously improve themselves through a connected and rewarding experience.

## Features

### 1. Learning Tracker

A comprehensive system for tracking learning progress across multiple topics and phases.

- **Topic Management** — Create, organize, and categorize learning topics by phase (Start, Restart, Explore)
- **Study Entries** — Log study sessions with duration, notes, and date tracking
- **Shared Collections** — Make topics public for community learning and bookmark shared topics from others
- **Progress Visualization** — Charts showing daily study duration, entry frequency, and topic-level breakdowns
- **Progress Flags** — Flag topics where you're struggling so the AI can offer targeted guidance
- **AI Progress Checks** — Get intelligent assessments of your learning consistency and recommendations for improvement
- **Phase Assignment** — Tag topics as Start (new skills), Restart (returning to paused skills), or Explore (curiosity-driven)

### 2. Fitness Tracker

A full-featured fitness companion for workouts, nutrition, and body composition tracking.

- **Fitness Profile** — Set weight, height, age, activity level, goal (lose/maintain/gain), diet type, and workout level
- **Workout Logging** — Track workouts by type with duration and estimated calories, including weight training fields (muscle group, sets, reps, load, workout split)
- **Food Logging** — Log meals with full macronutrient breakdown (protein, carbs, fat, fiber, sugar, calories) and meal categories
- **Weight Trend Chart** — Interactive weight progression chart with smart Y-axis scaling, proper date formatting, and single-data-point handling
- **TDEE Calculation** — Automatic Total Daily Energy Expenditure calculation based on your profile
- **AI Macro Estimation** — Instant food database lookup with AI fallback for estimating macros of any food item
- **AI Calorie Burn Estimation** — Instant exercise database lookup with AI fallback for estimating calories burned
- **Macro Nutrient Bars** — Visual progress bars showing daily protein, carbs, fat, fiber, and sugar intake vs. targets
- **Unit System Support** — Switch between metric and imperial measurements

### 3. Content Creation Tracker

A pipeline management system for content creators to plan, produce, and publish content.

- **Content Series** — Organize content into series with names, descriptions, and categories
- **Pipeline Stages** — Track content through idea → drafting → editing → published → archived workflow
- **Live Status Workflow** — Granular production status: not started → written → shoot → edit → posted
- **Content Types** — Support for blog posts, videos, and social media posts
- **Platform Tracking** — Tag content for YouTube, blog, Twitter, Instagram, and more
- **AI Script Review** — Get AI-powered feedback on your content scripts and drafts
- **AI Posting Time Optimization** — Receive data-driven recommendations for optimal posting schedules
- **Scheduling** — Set shoot dates, edit times, and post dates for production planning

### 4. Time Management

A productivity system with task management, focus sessions, and day planning.

- **Task Management** — Create, organize, and prioritize tasks with categories (work, personal, health, learning, other) and priority levels (low, medium, high)
- **Drag-and-Drop Ordering** — Reorder tasks intuitively with @dnd-kit drag-and-drop support
- **Focus Session Timer** — Dedicated focus mode with timer tracking for deep work sessions
- **Day Planner** — Plan your day with activities, rate your day (1–5), and write reviews
- **AI Task Classification** — Automatically classify tasks as productive or unproductive using AI
- **AI Productivity Ranking** — Get AI-powered ranking of your unproductive activities with replacement suggestions
- **AI Day Rating** — Receive an AI assessment of your day based on your activities and completion rate

### 5. Gamification System

A comprehensive progression system that makes consistency rewarding and engaging.

- **XP System** — Earn experience points for every activity across learning, fitness, content creation, and time management
- **Tiered Level Progression** — Levels require increasing XP (300 → 600 → 1000 → 2000 XP per level), creating a satisfying long-term growth curve
- **100+ Achievements** — Unlockable achievements across 4 categories (learning, fitness, time, content) with 4 tiers (bronze → silver → gold → platinum)
- **Daily Quests** — Personalized daily challenges with XP rewards to keep users engaged
- **Streak Tracking** — Current streak and longest streak tracking with visual badges
- **Community Leaderboard** — See how you rank against other users based on XP and activity
- **Achievement Badges** — Beautiful visual badges displayed on profiles showcasing unlocked achievements
- **Share Progress** — Optionally share achievements, fitness progress, content status, and learning progress with the community

- **Contextual Chatbots** — Specialized AI assistants for learning, fitness, content creation, and time management, each with domain-specific knowledge
- **Markdown Rendering** — Rich AI responses with proper markdown formatting (headings, lists, bold, code blocks, tables, and more)

## Architecture

```
┌─────────────────────────────────────────────────────────┐
│                      Client (Browser)                    │
│   React 19 · Next.js App Router · Tailwind CSS 4        │
│   shadcn/ui · Zustand · React Query · Recharts           │
└──────────────────────┬──────────────────────────────────┘
                       │ HTTP / WebSocket
┌──────────────────────▼──────────────────────────────────┐
│                   Next.js Server                         │
│   API Routes (60+) · Server Components · Middleware      │
│   NextAuth v4 · Rate Limiting · Abuse Detection          │
└──────────┬────────────────────────┬─────────────────────┘
           │                        │
┌──────────▼──────────┐  ┌─────────▼──────────────────────┐
│   PostgreSQL (Neon)  │  │   Pollinations AI (Free)       │
│   Prisma ORM (30+    │  │   GPT-4o → Mistral → Text      │
│   models)            │  │   No API key required           │
└──────────────────────┘  └────────────────────────────────┘
```

### Key Architectural Decisions

- **App Router with Route Groups** — `(auth)`, `(main)`, `(public)`, `admin` groups for clean layout separation
- **Server-First Rendering** — Server components by default, client components only when interactivity is needed
- **Middleware-Based Auth** — NextAuth v4 with middleware protecting authenticated routes, guest mode for public browsing
- **Resilient AI Layer** — 3-tier fallback ensures AI features always return a response even if primary models are down
- **Instant Local Lookup** — Food and exercise databases embedded in code for zero-latency macro/burn estimation with AI as fallback
- **Rate Limiting** — Category-specific rate limits (public, authenticated, AI, write, guest, admin) to prevent abuse
- **SEO Optimized** — JSON-LD schemas, Open Graph tags, Twitter cards, sitemap, and robots.txt for search engine visibility
- **PWA Ready** — Web manifest, theme color, and apple touch icon for installable progressive web app experience

## Getting Started

### Prerequisites

- **Node.js** 18+ or Bun
- **PostgreSQL** database (recommended: [Neon](https://neon.tech/) for serverless)
- **Vercel account** (for deployment) or any Node.js host

### Installation

```bash
# Clone the repository
git clone https://github.com/guvi4427-bot/sretracker-by-provolution-technologies-pvt-ltd.git
cd sretracker-by-provolution-technologies-pvt-ltd

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your database URL and auth secrets

# Set up the database
npx prisma generate
npx prisma db push

# Run the development server
npm run dev
```

The app will be available at `http://localhost:3000`.

### First-Time Setup

1. Create an account through the signup page
2. Complete the onboarding flow to set your phases and interests
3. Start tracking in any module — Learning, Fitness, Content, or Time
4. Earn XP and unlock achievements as you build consistency

## AI Integration

SRE uses **Pollinations AI** as its AI backend — a free, open-source AI proxy that requires no API key and has no rate limits.

### Architecture

```
User Request
    │
    ▼
┌─────────────────────────────────┐
│  1. Instant Local Lookup        │  ← Food DB / Exercise DB (0ms)
│     (macros, burn estimates)    │
└──────────┬──────────────────────┘
           │ Not found
           ▼
┌─────────────────────────────────┐
│  2. Pollinations AI (3-tier)    │  ← GPT-4o → Mistral → Text
│     25s timeout per tier        │
└──────────┬──────────────────────┘
           │ All tiers fail
           ▼
┌─────────────────────────────────┐
│  3. Fallback Response           │  ← Generic estimate
│     (never leaves user hanging) │
└─────────────────────────────────┘
```

### Features by Module

| Module | AI Capabilities |

## Deployment

### Vercel (Recommended)

The platform is optimized for Vercel deployment with:

- **Cron Jobs** — Daily learning inactivity check at 9:00 AM UTC via `vercel.json`
- **Edge-Optimized** — Middleware runs on the edge for fast auth checks
- **CDN Headers** — Custom caching headers for static assets (ads.txt, sitemap, robots.txt, favicon)
- **Security Headers** — X-Frame-Options, Content-Security-Policy, and more

```bash
# Deploy to Vercel
npm i -g vercel
vercel --prod
```

### Environment Setup on Vercel

Set the following environment variables in your Vercel project settings:

1. `DATABASE_URL` — Your PostgreSQL connection string
2. `POSTGRES_PRISMA_URL` — Pooled connection (recommended for serverless)
3. `NEXTAUTH_SECRET` — Random secret for JWT signing
4. `NEXTAUTH_URL` — Your production URL

### Alternative: Any Node.js Host

```bash
# Build the application
npm run build

# Start the production server
npm start
```

For detailed deployment instructions, see [DEPLOY.md](./DEPLOY.md).
